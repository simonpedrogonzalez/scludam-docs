scludam.pipeline moduleÔÉÅ

Module with default Detection and Membership Estimation Pipeline.

All steps of the process can be configured following the documentation for each function.

 3from scludam import DEP, CountPeakDetector, HopkinsTest, Query, RipleysKTest

 5# search some data from GAIA
 6# with error and correlation
 7# columns for better KDE
 8df = (
 9    Query()
10    .select(
11        "ra",
12        "dec",
13        "ra_error",
14        "dec_error",
15        "ra_dec_corr",
16        "pmra",
17        "pmra_error",
18        "ra_pmra_corr",
19        "dec_pmra_corr",
20        "pmdec",
21        "pmdec_error",
22        "ra_pmdec_corr",
23        "dec_pmdec_corr",
24        "pmra_pmdec_corr",
25        "parallax",
26        "parallax_error",
27        "parallax_pmra_corr",
28        "parallax_pmdec_corr",
29        "ra_parallax_corr",
30        "dec_parallax_corr",
31        "phot_g_mean_mag",
32    )
33    # search for this identifier in simbad
34    # and bring data in a circle of radius
35    # 1/2 degree
36    .where_in_circle("ngc2168", 0.5)
37    .where(("parallax", ">", 0))
38    .where(("phot_g_mean_mag", "<", 18))
39    # include some common criteria

41    .where_arenou_criterion()

45)
46
47# If data already has been downloaded, you can load it from a file:
48# > from astropy.table.table import Table
49# > df = Table.read("path_to_my_file/ngc2168_data.fits").to_pandas()
50
51
52# Build Detection-Estimation Pipeline
53dep = DEP(
54    # Detector configuration for the detection step
55    detector=CountPeakDetector(
56        bin_shape=[0.3, 0.3, 0.07],
57        min_score=3,
58        min_count=5,
59    ),
60    det_cols=["pmra", "pmdec", "parallax"],
61    sample_sigma_factor=2,
62    # Clusterability test configuration
63    tests=[
64        RipleysKTest(pvalue_threshold=0.05, max_samples=100),
65        HopkinsTest(),
66    ],
67    test_cols=[["ra", "dec"]] * 2,
68    # Membership columns to use
69    mem_cols=["pmra", "pmdec", "parallax", "ra", "dec"],
70).fit(df)
71
72# plot the results
73dep.scatterplot(["pmra", "pmdec"])
74# zoom on interesting area
75plt.axis([-1, 3, -5, 1])
76plt.show()
77
78# write results to file
79dep.write("ngc2168_result.fits")

Detection and membership estimation pipeline.

Class for running detection and membership estimation in a dataframe.

Variables:

  • detector (CountPeakDetector) ‚Äì Detector to use, it is required and should not be fitted.

  • det_cols (List[str]) ‚Äì Columns to use for detection. Should be a subset of the columns in the dataframe to be used.

  • sample_sigma_factor (float, optional) ‚Äì Factor to multiply the sigma of the detection region, in order to get a region sample for the overdensity detected, by default 1. Take into account that the sigma used is currently the bin shape in each dimension.

  • tests (List[StatTest], optional) ‚Äì Statistical tests to use for detection, optional, by default []. The list can include a non fitted instance of StatTest.

  • test_cols (List[List[str]], optional) ‚Äì Columns to use for statistical tests, optional, by default []. Note that the list must have the same length as the tests list, and each item is a list of columns to use for the statistical test.

  • test_mode (str, optional) ‚Äì Mode to use for statistical tests, optional, by default ‚Äòany‚Äô. If ‚Äòany‚Äô, the test is considered passed if any of the stat tests results in the rejection of their null hypothesis. Other options are ‚Äòall‚Äô and ‚Äòmajority‚Äô.

  • clusterer (SHDBSCAN, optional) ‚Äì Clusterer to be used to get the initial probabilities. By default, an instance of SHDBSCAN with the following parameters: auto_allow_single_cluster=True, noise_proba_mode=‚Äùconservative‚Äù, cluster_proba_mode=‚Äùhard‚Äù, scaler=RobustScaler() (from sklearn.preprocessing).

  • estimator (DBME, optional) ‚Äì Estimator to use for membership estimation, by default an instance of DBME with the default parameters.

  • mem_cols (List[str], optional) ‚Äì Columns to use for membership estimation, by default, det_cols. Should be a subset of the columns in the dataframe to be used. It is recommended to use the same columns as for detection, or at least some of them, so center estimation can be used for better clustering. For the estimation process, error and correlation columns can be used if they are available in the dataframe. For example, if mem_cols is [‚Äòx‚Äô, ‚Äòy‚Äô], the program will check if columns ‚Äòx_error‚Äô, ‚Äòy_error‚Äô, ‚Äòx_y_corr‚Äô are present.

  • n_detected (int) ‚Äì Output attribute, number of overdensities detected.

  • detection_result (DetectionResult) ‚Äì Output attribute, result of the detection.

  • test_results (List[TestResult]) ‚Äì Output attribute, list of statistical test results.

  • is_clusterable (List[bool]) ‚Äì Output attribute, list of decisions taken in the tests, whether the overdensity can be clustered.

  • n_estimated (int) ‚Äì Output attribute, number of overdensities estimated, that is, overdensities that passed the tests.

  • proba (np.ndarray) ‚Äì Output attribute, membership probabilities of each cluster found.

  • labels (np.ndarray) ‚Äì Output attribute, label of each data point, starting from -1 (noise).

  • limits (List[Numeric1DArray]) ‚Äì Output attribute, list of limits used for each detection region.

  • masks (List[NDArray[bool]]) ‚Äì Output attribute, list of masks used for each detection region.

  • clusterers (List[SHDBSCAN]) ‚Äì Output attribute, list of clusterers used for each detection region.

  • estimators (List[DBME]) ‚Äì Output attribute, list of estimators used for each detection region.
